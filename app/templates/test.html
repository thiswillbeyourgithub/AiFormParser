LLM benchmarking

    

      Sweeps thread count (1, 3, and the value the user pipeline picks on
      this device) against compute offload (all GPU, all CPU, GPU with the
      vision encoder forced to CPU). For each combination, the model is
      reloaded, a short text generation runs and a synthetic multimodal OCR
      runs, each stopped after 10 generated tokens (thinking included).
      TTFT and tok/s are recorded per task. The final table is sorted by
      multimodal tok/s, descending, since OCR is the production workload.
      Uses the Model picker and the Model / Completion options textareas
      above. Cancel interrupts after the current combination's load.
