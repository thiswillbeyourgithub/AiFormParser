      Each preset bundles a model name plus the load and sample parameters
      a researcher should default to. Researchers pick a preset from the
      list and can override individual parameters before running. The
      preset marked default is pre-selected for them. A survey with no
      preset cannot be processed.

Self-hosted models

    

      To self-host weights, create one subdirectory per model under
      MODELS_DIR on the data volume (e.g.
      MODELS_DIR/qwen3-4b/model.gguf) and place the matching
      *mmproj*.gguf projector next to it for multimodal support.
      A flat .gguf at the top of MODELS_DIR still
      works but has no projector. There is intentionally no upload endpoint.
    

    

      Avoid IQ imatrix quants (filenames containing IQ1_*,
      IQ2_*, IQ3_*, IQ4_*). Per the
      wllama
      README these are significantly slower than the classic K-quants
      (Q4_K_M, Q5_K_M, ...) and are not recommended.
      Any IQ-quant file listed below is tagged inline.
